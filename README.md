# Auerbach Lab Natural Language Processing

### NLP Code for the Translational Research on Affective Disorders and Suicide Laboratory 

This repository includes data cleaning and preprocessing code for several projects using passively collected keyboard inputs via the Effortless Assessment Research System (EARS) app.
